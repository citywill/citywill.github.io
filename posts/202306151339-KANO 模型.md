# KANO 模型

KANO 模型是用户需求分类和优先排序工具，以分析用户需求对用户满意的影响为基础，体现了产品性能和用户满意之间的非线性关系。

根据不同类型的质量特性与顾客满意度之间的关系，狩野教授将产品服务的质量特性分为五类：

1. 基本（必备）型质量：Must-be Quality/ Basic Quality。理所应当的因素，没有不行，但满足不会引起客户额外的满意度。
2. 期望（意愿）型质量：One-dimensional Quality/ Performance Quality。需求满足和客户满意成正比关系。
3. 兴奋（魅力）型质量：Attractive Quality/ Excitement Quality。不提供客户不会不满，但提供后会有效提升客户满意度。
4. 无差异型质量：Indifferent Quality/Neutral Quality。可有可无，对满意度没有影响的因素。
5. 反向（逆向）型质量：Reverse Quality，亦可以将 'Quality' 翻译成“质量”或“品质”。引起不满的因素。

## 需求类型的满足和用户满意度的关系

|        | 满足 | 不满足 |
| ------ | ---- | ------ |
| 基本   | -    | ↓      |
| 期望   | ↑    | ↓      |
| 魅力   | ↑↑   | -      |
| 无差异 | -    | -      |
| 反向   | ↓    | ↑      |
